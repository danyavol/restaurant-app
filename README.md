# restaurant-app
## Project setup
### 1. Download and install **Node.js** and **npm**
npm is distributed with Node.js which means that when you download Node.js, you automatically get npm installed on your computer.
[Download Node.js](https://nodejs.org/en/download/)

### 2. Download and install to your PC **MongoDB server**
After installation, Mongo server will automatically start on port 27017.
[Download MongoDB](https://www.mongodb.com/try/download/community)
### 3. Install **node_modules**
Navigate to folder **restaurant-app/client** and run command
```
npm install
```
Do the same at folder **restaurant-app/server**
### 4. Launch project
Navigate to folder **/server** and run
```
npm start
```
After do the same at folder **/client**
